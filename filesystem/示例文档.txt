喜碧电子阅读器使用说明

喜碧电子阅读器！专为墨水屏优化的

== 主要特性 ==

1. 高对比度设计
   - 纯黑白配色方案
   - 清晰的边框和分割线
   - 适合墨水屏显示

2. 大字体友好
   - 默认18px字体大小
   - 支持字体大小调节（12px-32px）
   - 优化的行距设置

3. 简洁的界面
   - 最小化的UI元素
   - 专注于内容阅读
   - 减少不必要的视觉干扰

== 使用方法 ==

文件浏览：
- 点击文件夹图标进入子目录
- 点击"返回上级"回到父目录
- 文本文件显示"阅读"按钮
- 所有文件都可以下载

阅读功能：
- 🔤- / 🔤+ : 调整字体大小
- 📏 : 切换行距（1.4-2.2倍）
- 📐 : 切换阅读宽度（窄屏/宽屏）
- 进度条显示阅读进度

快捷键：
- Ctrl + ↑/↓ : 调整字体大小
- Ctrl + Space : 调整行距
- Ctrl + W : 切换宽度模式
- Home/End : 跳转到文档开头/结尾
- F1 或 Shift+? : 显示帮助

== 技术特点 ==

后端：Flask轻量级框架
前端：纯HTML+CSS+JavaScript
优化：专门针对墨水屏的显示特性
兼容：支持UTF-8和GBK编码
安全：路径安全检查，防止目录遍历

== 支持的文件类型 ==

文本文件：
- .txt - 纯文本文件
- .md - Markdown文档
- .py - Python源代码
- .js - JavaScript文件
- .html - HTML文档
- .css - CSS样式表
- .json - JSON数据文件
- .xml - XML文档
- .csv - CSV表格文件
- .log - 日志文件

其他文件：
- 非文本文件提供下载功能
- 自动检测文件编码
- 支持中文文件名

== 墨水屏优化说明 ==

1. 颜色方案：
   - 纯黑色文字 (#000)
   - 纯白色背景 (#fff)
   - 浅灰色辅助区域 (#f8f8f8)

2. 字体选择：
   - 优先使用Times New Roman
   - 中文使用SimSun（宋体）
   - 保证在墨水屏上的清晰度

3. 布局设计：
   - 大按钮，易于触摸
   - 清晰的边框分割
   - 充足的内边距和外边距

4. 交互优化：
   - 减少动画效果
   - 简化hover状态
   - 专注于内容展示

== 安装和运行 ==

1. 安装依赖：
   pip install -r requirements.txt

2. 运行服务器：
   python app.py

3. 访问地址：
   http://localhost:5000

4. 局域网访问：
   http://[你的IP地址]:5000

== 注意事项 ==

- 建议在墨水屏设备上使用单色模式
- 可以调整浏览器的缩放比例适配屏幕
- 支持触摸和键盘操作
- 自动保存阅读位置和个人设置
